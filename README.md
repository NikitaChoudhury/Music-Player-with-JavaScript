# Music-Player-with-JavaScript
FUNCTIONS:

1.	Play/Pause song.
2.	Random song.
3.	Previous song.
4.	Next song.
5.	Repeat song.

This Music Player is built using HTML,CSS and JavaScript.

The layout changes with each song.
